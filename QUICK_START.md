# 🚀 快速配置指南

## 第一步：获取 GLM-4.7 API Key

1. 访问 [智谱AI开放平台](https://open.bigmodel.cn/)
2. 注册账号并完成实名认证
3. 进入控制台 → API Key
4. 创建新的 API Key
5. 复制 API Key

## 第二步：获取高德地图 API Key

1. 访问 [高德开放平台](https://lbs.amap.com/)
2. 注册成为开发者
3. 进入控制台 → 应用管理 → 创建新应用
4. 添加 Key，选择 "Web端(JS API)"
5. 复制 Key 和安全密钥（如有）

## 第三步：配置环境变量

在项目根目录下找到 `.env` 文件，填入您的 API Key：

```env
VITE_GLM_API_KEY=你的GLM_API_KEY
VITE_AMAP_API_KEY=你的高德地图API_KEY
```

## 第四步：重启开发服务器

```bash
# 停止当前服务器 (Ctrl+C)
# 然后重新启动
npm run dev
```

## 完成！🎉

现在您可以：
1. 填写旅行信息，点击"生成行程计划"
2. 查看由 GLM-4.7 AI 生成的智能行程
3. 在高德地图上查看景点位置和游览路线

## 常见问题

### Q: 不配置 API Key 能用吗？
A: 可以！应用会使用模拟数据作为降级方案，但功能有限。

### Q: 如何测试配置是否成功？
A: 生成行程后，如果没有看到错误提示，说明配置成功。

### Q: API 调用失败怎么办？
A: 
1. 检查 API Key 是否正确
2. 确认账号是否有余额（GLM-4.7）
3. 查看浏览器控制台错误信息
4. 重启开发服务器
